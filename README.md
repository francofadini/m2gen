# M2Gen

MVP base project code generator. Built with Deno@1.3.0

## Install

 `deno install --allow-run --name m2gen src/main.ts`

### Usage

Run `m2gen` and follow the instructions

### Prerequisites

- [Deno](https://deno.land/)
